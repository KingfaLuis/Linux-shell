# 3 正则其他字符的表示方式

- 边界字符，或者说是位置字符

  字符"^"和"$"分别用于匹配行首和行尾

- 元字符

  - \w:匹配任何字符类型，包括下划线([A-Za-z09_])
  - \W:匹配任何非字符类型.([\^A-Za-z09_])
  - \b:代表单词的分隔



# 4 正则组合，逻辑字符表示



| 元字符 |           描述           |
| :----: | :----------------------: |
|   *    |     重复0次或更多次      |
|   +    |     重复一次或更多次     |
|   ？   |      重复0次或一次       |
|  {n}   |         重复n次          |
|  {n,}  |     重复n次或更多次      |
| {n,m}  | 重复不少于n次，不多于m次 |
